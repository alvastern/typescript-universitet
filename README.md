# TypeScript Universitet
Detta är en projektuppgift i kursen Programmering i TypeScript som har gått ut på att jag som student har utvecklat en webbapplikation som lagrar kurser från Mittuniversitetet. Dessa kurser ska lagras i en lista på en undersida, här kan användare även söka efter kurser, filtrera kurser och sortera kurser. 

Det finns en andra undersida som innehållet ett personligt ramschema för studenterna. I detta ramschema kan användare lägga till kurser från kurs-listan, se det totala antalet högskolepoäng, samt ta bort kurser från kurslistan.
